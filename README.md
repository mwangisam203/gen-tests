# Python Learning Lab

This repo is a small playground for practicing Python by reading, running, and
testing short examples.

## How to Use It

Run a lesson directly:

```bash
python -m learning_lab.cli numbers
```

Run the tests:

```bash
python -m pytest
```

## Lesson Path

1. Numeric functions
2. List and dictionary patterns
3. Text cleanup
4. File reading and writing
5. CSV data
6. Simple web parsing
7. NumPy arrays
8. Command-line programs
