# Lesson Plan

Use this path when you want a quick practice session.

## 1. Read

Pick one module in `learning_lab/` and read the functions from top to bottom.

## 2. Predict

Before running the tests, guess what each assertion should prove.

## 3. Run

Run the related test file with:

```bash
python -m pytest tests/test_numbers.py
```

## 4. Change

Make one tiny change, run the test again, and notice which behavior moved.

## 5. Explain

In your own words, explain:

- What data went into the function
- What data came out
- What error cases were handled

## Second Chapter

After the first pass, choose one of the newer modules and add one extra test
before changing the implementation. That habit teaches you to describe behavior
before you edit behavior.
